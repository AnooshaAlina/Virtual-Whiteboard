# 🖐️ Virtual AI Whiteboard

A real-time **gesture-controlled virtual whiteboard** built using **OpenCV** and **MediaPipe**. This application allows users to draw, erase, and create shapes in the air using hand gestures — no physical input device required.

---

## 🚀 Features

- ✍️ Draw using index finger
- 🧽 Erase using open palm gesture
- 📐 Draw shapes (rectangle) using two fingers
- 🎯 Select colors via pinch gesture
- 🎨 Multiple brush sizes and color palette
- ↩️ Undo functionality
- 🗑️ Clear canvas option
- 🧊 Frosted live camera background with overlay canvas
- 🖐️ Real-time hand tracking using MediaPipe

---

## 🧠 Technologies Used

- **Python**
- **OpenCV** – for image processing and UI rendering
- **MediaPipe** – for hand tracking and gesture recognition
- **NumPy** – for efficient array operations
